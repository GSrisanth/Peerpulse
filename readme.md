# PeerPulse

PeerPulse is an interactive web application designed to help users understand and evaluate peer pressure through a self-assessment quiz.

## Features
- Interactive flip cards for learning concepts
- Self-assessment quiz
- Real-time score analysis
- Clean and modern UI

## Tech Used
- HTML
- CSS
- JavaScript

## How to Run
1. Open index.html in browser  
or  
2. Run using local server:
   python3 -m http.server 8000
